# Movie Recommendation System

## Description
This project is a Movie Recommendation System based on content similarity.
It recommends movies similar to the movie selected by the user.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Machine Learning Concepts
- Unsupervised Learning
- TF-IDF Vectorization
- Cosine Similarity

## Files
- movies.csv : Dataset
- movie_recommender.ipynb : Main project file

## How to Run
1. Install required libraries
2. Open movie_recommender.ipynb
3. Run cells step by step
4. Call recommend("Movie Name")
